A ramble on two horses.

About a month ago, I learned about the two-horse rule. It goes like this: if one horse can pull 700 pounds and another horse can pull 800 pounds, how much weight will they pull when yoked together?

The arithmetic answer would be 1500 pounds. But two horses, when yoked together, can pull 3000 pounds. Huh?

Synergy: the cooperation of two or more agents to produce a combined effect greater than the sum of their separate effects.

I've seen synergy at work when I run. On my own, I can be an unmotivated bum. But when I run with my fellow bum...

13% improvement when two bums run together.
